# KJELDGAARD Docs

Denne mappe indeholder master-dokumenter som bruges af:
- Claude Projects (henter via API)
- Railway Ad Factory (serverer via API)
- Team reference

## Filer
Upload dine FACTS-filer, DOS_AND_DONTS, SALES_CORE osv. hertil.
De bliver automatisk tilgængelige via API:
`GET /api/docs` — liste alle docs
`GET /api/docs/:filename` — hent specifikt dokument
